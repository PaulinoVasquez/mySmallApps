Sobre la sintaxis de las expresiones

  • El metodo empleado para la evaluacion de expresiones se basa en el RPN(Reverse Polish notation) y sin embargo no cuenta con un analizador lexicografico para el reconocimiento de expresiones. Es por eso que se confia en que el usuario ingrese correctamente la expresion para evaluarla. Dentro de un tiempo espero yo no muy largo, debere incluir el parser para evitar errores espantosos como los que pueden ocurrir en este momento.
