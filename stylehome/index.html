Editor de Estilos para mi casa
El uso de este software es relativamente sencillo, la parte izquierda simula la perspectiva de frente de la casa, donde hay cuatro ventanas y una puerta. Ahora, la parte derecha contiene dos divisiones, una que es especificamente para la barda en general de la casa, y la otra para unicamente la parte borduda de este enjarre. Contiene tambien tres entradas para cada division, para los tres colores principales del RGB (RVA en español) los cuales podrán realizar las convinasiones necesarias de los 1.6581375x107 colores disponibles.
